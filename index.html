🛍 メルカリ出品アシスタント

1商品写真を追加

ここに写真をドラッグ＆ドロップ
またはクリックして選択（最大10枚・1枚目がメイン写真）

2商品情報（わかる範囲でOK）

3AIで写真から分析

「写真を分析」はAPIキー設定が必要（右上⚙）。キーが無い場合は「プロンプトをコピー」して、Claudeアプリに写真と一緒に貼り付けても同じ結果が得られます。

4売れた価格を調べる（実売価格を優先）

開いたページで「自分の商品と状態が近いもの」の売れた価格を3〜5件メモして、下に入力してください。

5出品下書き（メルカリに貼り付け）

0 / 40
0 / 1000
メルカリアプリで「出品」→ 写真を追加 → 貼り付け → 最後に「下書きに保存」を押せば下書き完成です。

Claude API 設定（任意）

APIキーを設定すると「写真を分析」ボタンでAIが直接、状態判定・タイトル・説明文を生成します。キーは端末内（localStorage）のみに保存されます。
取得先：console.anthropic.com → API Keys

⚠️ 家族共用・他人のパソコンではキーを保存しないでください。使い終わったら「削除」ボタンで消せます。
